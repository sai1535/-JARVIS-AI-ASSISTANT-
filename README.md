# 🤖 JARVIS - Python Voice Assistant

A simple AI-powered Voice Assistant built using Python that can listen to voice commands, respond with speech, open websites, launch Windows applications, tell jokes, and perform basic assistant tasks.

## 🚀 Features

- 🎤 Voice Recognition using SpeechRecognition
- 🔊 Text-to-Speech using pyttsx3
- 🌐 Open Websites
  - Google
  - YouTube
  - Instagram
- 🔍 Search anything on Google
- 😂 Tell random jokes
- 📝 Open Notepad
- 🧮 Open Calculator
- 🕒 Tell the current time
- 👋 Greeting and Exit commands

---

## 🛠️ Technologies Used

## 📦 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/Jarvis-Voice-Assistant.git
```

### 2. Move into the Project Folder

```bash
cd Jarvis-Voice-Assistant
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

Or install manually:

```bash
pip install pyttsx3
pip install SpeechRecognition
pip install PyAudio
pip install pyjokes
```

---

## ▶️ Run the Project

```bash
python jarvis.py
```

## 🎙️ Supported Voice Commands

| Command | Action |
|----------|--------|
| Hello | Greets the user |
| Time | Tells current time |
| Open Google | Opens Google |
| Open YouTube | Opens YouTube |
| Open Instagram | Opens Instagram |
| Search Python | Searches Google |
| Tell me a joke | Speaks a random joke |
| Open Notepad | Opens Notepad |
| Open Calculator | Opens Calculator |
| Bye / Exit | Closes Jarvis |

## 💡 Example

```
You: Open Google

You: What is the time?
JARVIS: The time is 10:35 PM

You: Tell me a joke
JARVIS: (Random joke)

You: Bye
```
